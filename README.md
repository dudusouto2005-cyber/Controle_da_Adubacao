# Controle_da_Adubacao
O controle da adubação foi realizado em uma área experimental de 42 m² cultivada com tomate da variedade San Marzano.

Foram registradas quatro aplicações do fertilizante NPK 04-14-08, totalizando 520 kg/ha.

O acompanhamento dessas aplicações permite verificar se o manejo nutricional está sendo executado conforme o planejamento, evitando tanto a deficiência quanto o excesso de nutrientes.

Além disso, manter registros das aplicações facilita o planejamento das próximas adubações e contribui para maior eficiência no uso dos fertilizantes.
